## ZSH

### Install

Install with homebrew

```
$ brew install zsh
```

### Set ZSH as the default shell

To be able to use this shell, add its full pathname to `/etc/shells`

```
/usr/local/bin/zsh
```

To enable this new shell, fire the following command:

```
$ chsh -s /usr/local/bin/zsh
```
